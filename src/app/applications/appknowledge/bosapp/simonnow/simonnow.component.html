SIMONNOW


Simon Now holds the Agreed upon rates from customer.The validation Rules will be set as per customer requirements How the rates must be calculated How The Invoice gets Delivered.All the validations and checks can be automated, and the Agreed upon Rates will be automatically sent to RBI during Rating

Simon Now is a Finance Application in which the rates and accessorial charges that are assigned to a particular Publication/Bill To Code/Lanes are displayed. This application contains all the information concerning the rates and accessorial charges. The published data in Simon Now provides the basis for rating processes. PDM/PDA ensures that the information within the Simon Now is accurate and up-to-date. Lane rates can be entered into the system through manual entry, ePublish and file/Eagle uploads. Accessorial rates can only be entered manually.

There are different type of Search options in Simon Now. They are:

1. Publication Code

2.Bill To Code

3.Lanes

4.Mixed Code List

Every serach criteria will be pulling the details of the Publication, Sections assigned in it, Items which are present under each section, Lanes which are contained in each Item, Accessorials assigned under each Lane and then so on..
