ACCESSORIAL AUTHORIZATION AND BILLING (AXB)

The main use of the application is to Authorize and Bill the Accessorial Charges. This application will look for loads with accessorial charges such as Power Detention, Dwell Detention, DRVRLOAD, DRVRUNLD, LUMPLOAD, LUMPUNLD for billing and authorization. It pulls the Accessorial rules from the CCI and Authorization rules from Simon Now. There are 3 tabs in the application: Authorization, Billing and Search.


The Accessorials will be setup originally in Simon Now and all the rates and charges which are required for the Billing process will be picked up from the Simon Now Application.

Billing Tab: This tab maintains all the billing processes happening and as soon as we click on the Billing tab we will get the below screen in which it displays the Next Bill Load.Then it will be displaying the order details which ever needs to be billed. If there are no loads present to bill it will be displaying ‘No Loads to be billed’ message.


The Billing User would be able to perform the below activities:

1.Read details of the order

2.Update AutoRate Flag

3.View List Number

4.Add/View order comments

5.View customer Information

6.Add standard order comments when any workaction is selected by the user.


Authorization Tab: The things which should be done in Authorization tab are:

1.Checking the details of the order.

2.Viewing the Filenet Documents necessary for billing.

3.Add Order Comments.

4.View Order Comments.

5.View Customer Details.

6.View Load Reference Number

7.View Error messages

8.View Publication Information

9.Adding Standard Order Comments by the system when the order is Completed or Denied.

10.View OBC comments(Only if the charge code is DetentionP)
